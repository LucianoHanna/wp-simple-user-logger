Logs all user-related activities in WordPress including logins, updates, and role changes.

== Description ==

WP Simple User Logger creates detailed logs of all user-related activities in your WordPress installation. It tracks:

* Successful and failed login attempts
* User creation
* Profile updates
* User deletions
* Password resets
* Role changes

All logs are stored in a file within the WordPress uploads directory.

== Installation ==

1. Upload the 'wp-simple-user-logger' folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Logs will automatically be created in your uploads directory
